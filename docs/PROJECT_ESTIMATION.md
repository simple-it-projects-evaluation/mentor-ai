# Оценка разработки mentor.ai

## Вводные данные

| Параметр | Значение |
|----------|----------|
| **Срок MVP** | 1 месяц (22 рабочих дня) |
| **Подход** | Claude Code + готовые SaaS-решения |
| **Текущий статус** | Frontend прототип готов на 70% |

---

## Преимущества работы с нами

| Преимущество | Описание |
|--------------|----------|
| **AI-ускоренная разработка** | Используем Claude Code — сокращение сроков в 2-3 раза по сравнению с классической разработкой |
| **Прототип уже готов** | 70% UI реализовано до старта основной разработки — быстрый запуск |
| **Оплата по этапам** | 6 этапов с чёткими критериями приёмки — платите только за результат |
| **Минимум кастомной инфраструктуры** | Используем проверенные SaaS (Supabase, Vercel, Deepgram) — меньше багов, быстрее запуск |
| **Фиксированный срок MVP** | 1 месяц — без растягивания сроков |
| **Прозрачная оценка** | Детальная разбивка по блокам — понятно, за что платите |
| **Опыт с AI-интеграциями** | Claude API, RAG, Speech-to-Text — не экспериментируем на вашем проекте |
| **Полный цикл** | От прототипа до продакшена одной командой — без потери контекста |

# Раздел 1: MVP

## Функционал MVP (из ТЗ)

- Анализ звонков по 5 этапам продаж
- Определение успешных/неуспешных звонков
- Выявление потерянных клиентов
- Рекомендации по возврату клиентов
- Речевой тренажёр (голосовой)
- Аналитика по менеджерам
- Расчёт конверсии и денег
- Контроль выполнения рекомендаций
- Уведомления о саботаже
